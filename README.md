# 📊 Interactive Sales Performance Dashboard (Microsoft Excel)

## Project Overview

This project demonstrates an end-to-end sales analysis workflow in Microsoft Excel, from raw data preparation to an interactive dashboard. The dashboard provides business insights through KPIs, PivotTables, PivotCharts, and Slicers, enabling users to analyze sales performance efficiently.

---

## Project Objectives

- Clean and prepare raw sales data
- Perform exploratory sales analysis
- Build interactive PivotTables and PivotCharts
- Design a professional dashboard
- Present business insights for decision-making

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas
- Conditional Formatting

---

## Dashboard Features

The dashboard includes:

- ✅ Total Sales KPI
- ✅ Total Quantity Sold
- ✅ Total Orders
- ✅ Average Sales
- ✅ Regional Sales Analysis
- ✅ Monthly Sales Trend
- ✅ Sales by Category
- ✅ Top 5 Products
- ✅ Top 10 Sales Representatives
- ✅ Interactive Slicers for filtering

---
